# 📱 ETEC - Biografia Tecnológica

Projeto desenvolvido em sala de aula na disciplina **DESENVOLVIMENTO PARA DISPOSITIVOS MÓVEIS I**, com o objetivo de criar um aplicativo simples em **React Native** apresentando a biografia de uma personalidade relevante na área da tecnologia.

## 📌 Objetivo

Construir uma tela de aplicativo utilizando os componentes básicos do React Native, aplicando os conceitos aprendidos em aula.

## ✅ Requisitos do Projeto

- [x] Uso dos componentes básicos:
  - **Text**
  - **View**
  - **Image**
  - **ScrollView**
- [x] Estrutura organizada e estilizada com **StyleSheet**
- [x] Conteúdo biográfico de uma personalidade tecnológica
- [x] Criatividade na apresentação

## 🛠️ Tecnologias Utilizadas

- **React Native**
- **Expo**
- **JavaScript**
